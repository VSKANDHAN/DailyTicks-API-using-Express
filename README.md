# API END POINTS
## get: /api/tasks
Fetch All the secrets from the MongoDB 
## post: /api/tasks
body:{"title":"","description":""}
Post a new secret to the MongoDB
## patch: /api/tasks/:id 
body:{"title":"","description":""}
Update a particular secret in MongoDB
## delete: /api/tasks/:id 
Delete a particular secret in MongoDB
